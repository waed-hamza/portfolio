E-COMMERCE APP

❮ ❯

E-Commerce application built to enable online shopping to anyone at any time, By showing the goods to the buyer. It is the final project I did in the fundemental course with Gasa sky geeks.

This website consists of :

  1. Four Pages : Landing page , Customer page , Seller page and Cart page.
  2. The project supports responsive, so it can be displayed on many devices.
  3. DOM manipulation, Local Storage and other features are used.
  4. Other features and filters by price or category are provided.

This project provides services to purchase goods and controls the basket if they want to keep or delete the goods. As for the store owner, he can add new goods and delete any goods.
